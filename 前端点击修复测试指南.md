
### 步骤3：测试点击功能

2. 在右侧问题区域输入问题
3. 点击"确认"按钮
4. 观察左侧聊天区域是否显示：

### 步骤4：检查浏览器控制台

如果点击仍无响应，按 F12 打开开发者工具：

1. **Console 标签页**：查看是否有JavaScript错误
2. **Network 标签页**：
   - 点击确认按钮后
   - 查看是否有请求发送到 `http://localhost:1090/chatbot/api/v1.0/get`
   - 检查请求状态码（应为200）

## 常见问题排查

### 问题1：点击按钮无任何反应

**检查项：**
```javascript
// 打开浏览器控制台，输入：
document.querySelector('.submit')
// 如果返回 null，说明按钮选择器有问题
```

**解决：** 确认按钮有 `class="submit"` 属性

### 问题2：Network Error 或 CORS 错误

**检查项：**
- 后端是否在 1090 端口运行
- 是否看到 `Access-Control-Allow-Origin` 错误

**解决：**
```powershell
# 确认 flask-cors 已安装
pip show flask-cors

# 如果未安装
pip install flask-cors==4.0.0
```

### 问题3：请求超时

**检查项：**
- 后端是否正在处理请求（查看后端控制台输出）
- 是否有网络问题

**解决：**
- 检查后端日志
- 确认API密钥配置正确
