# 使用本地BGE-small-zh模型说明

## 🎉 改造完成

您的项目已从依赖OpenAI API改为使用本地BGE-small-zh模型！

---

## 📋 需要执行的步骤

### 1. 安装依赖

```bash
pip install sentence-transformers==2.7.0
```

### 2. 重新生成数据库embedding

**重要！**数据库中的embedding是用OpenAI模型生成的（3072维），需要用BGE模型重新生成（512维）：

```bash
cd e:\组里的项目\chatbot2024
python regenerate_embeddings.py
```

### 3. 测试模型

```bash
python test_bge.py
```

### 4. 启动服务

```bash
python app.py --port 1090
```

## ⚠️ 注意事项

### 关于GPT API

目前`gpt.py`仍使用OpenAI GPT API生成最终回答。如果您也想替换为本地模型或Qwen API，请告诉我！

### 模型下载

首次运行时，BGE-small-zh模型会自动从HuggingFace下载（约33MB）。如果网络受限，可以：

1. 使用国内镜像：
   ```bash
   export HF_ENDPOINT=https://hf-mirror.com
   ```

2. 或手动下载模型到本地
